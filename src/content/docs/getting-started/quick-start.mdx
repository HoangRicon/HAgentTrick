# Quick Start

Get up and running with this quick start guide.

## Create Your First Page

Creating a new page is easy. Just create a new file in the `src/app` directory:

```typescript
// src/app/hello/page.tsx
export default function HelloPage() {
  return (
    <main>
      <h1>Hello, World!</h1>
      <p>Welcome to my documentation site.</p>
    </main>
  );
}
```

## Add Documentation

Add new documentation by creating MDX files in the `src/content/docs` directory:

```mdx
---
title: My New Page
description: A description of my page
---

# My New Page

This is my new documentation page!
```

## Customize the Theme

The site supports both light and dark mode. Users can toggle between themes using the button in the header.

## Deploy

When you're ready to deploy, run:

```bash
npm run build
```

This will generate a static site in the `out` directory, ready to be deployed to GitHub Pages or any static hosting provider.
