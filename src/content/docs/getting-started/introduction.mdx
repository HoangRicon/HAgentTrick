# Introduction

Welcome to our documentation! This guide will help you get started with our platform.

## What is This?

This is a modern documentation website built with Next.js, featuring:

- **Fast Performance** - Static site generation for blazing fast page loads
- **Beautiful Design** - Clean, modern UI with dark mode support
- **Easy Navigation** - Intuitive sidebar and search functionality
- **Code Highlighting** - Beautiful syntax highlighting for code blocks

## Getting Started

To get started, check out the [Installation](/docs/getting-started/installation) guide to set up your development environment.

## Features

### Markdown Support

Write your documentation in Markdown with full support for:

- Headings (H1-H6)
- **Bold** and *italic* text
- Lists (ordered and unordered)
- Links and images
- Code blocks with syntax highlighting

### MDX Components

You can also use React components in your Markdown files:

```jsx
<Button variant="primary">Click Me</Button>
```

### Dark Mode

Toggle between light and dark mode using the button in the header.
